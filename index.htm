Vim Cheat Sheet

Movement

  • hleft
  • jdown
  • kup
  • lright
  • Hhigh
  • Mmiddle
  • Llow
  • <ctrl> uup
  • <ctrl> ddown
  • eend of word
  • Eend of word (jump punctuation)
  • wbeginning of word
  • Wbeginning of word (jump punctuation)
  • bback to end of word
  • Bback to end of word (jump punctuation)
  • f/F charfind char forward/back
  • t/T chartill char forward/back
  • ; & ,repeat forward/back t/T & f/F
  • /search
  • n/Nrepeat search forward/back
  • <ctrl> ^beginning of line
  • <ctrl> $end of line
  • {/}jump forward/back paragraph (blank line)

Edit

  • ciwclear inner word
  • citclear inner tag
  • vipselect inner paragraph
  • ci surrounding-charchange inner surrounding-char
  • dawdelete a word
  • dipdelete inner paragraph
  • dapdelete around paragraph
  • >ipindent inner paragraph
  • .repeat indent
  • =ipre-align inner paragraph
  • o/Onew line below/above

File

  • :eopen file
  • :split / :vsplitsplit current file

Tab

  • :tabnewtab new
  • gt/gTgoto next/previous tab
  • <ctrl>-w Tbreak current window out to new tab

Window

  • :new / :vnewsplit
  • :split / :vsplitsplit current file
  • <ctrl>-w JK/HLmove window
  • <ctrl>-w +-/<>/=resize window
